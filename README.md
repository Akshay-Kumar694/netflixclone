# netflixclone

Netflix Clone – Project Description

This project is a front-end clone of the Netflix landing page built using HTML, CSS, and Bootstrap. It replicates the main design and layout of Netflix’s homepage, showcasing modern UI elements and responsive design.

Key Features:
Navigation Bar: Includes the Netflix logo, language selector, and Sign In button.
Hero Section: Highlights the main promotional text with a call-to-action input box for email and a “Get Started” button.
Feature Sections: Displays Netflix features such as:

Watch on TV and devices with embedded video.
Download shows for offline viewing.
Watch anywhere on multiple devices.
Create kid-friendly profiles.

FAQ Section: Accordion-style boxes displaying common user questions.
Footer Section: Contains multiple columns of helpful links and a contact number.

Technologies Used:

HTML5 – for structuring the page
CSS3 – for custom styling (via style.css)
Bootstrap 5 – for responsive layout and prebuilt components
Google Fonts – for attractive typography

Purpose:

The Netflix Clone serves as a front-end design project to practice responsive web development, layout design, and multimedia integration (images & videos).
